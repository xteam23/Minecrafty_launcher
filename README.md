## Minecrafty - A Cracked Minecraft PvP Launcher for terminal users

Minecrafty is a free and open-source luancher for terminal users who dont wanna buy minecraft.

## How to use

First you need python for this (I'm sure all linux machines come with python pre-installed)

Then make a virtual enviroment using python.
If you dont know how to, refer below:

`python3 -m venv env`

It will take a few seconds, and when it finishes you have to get in the enviroment.
To get in look below:

`source env/bin/activate`

Then get out of the enviroment. To get out, refer below:

`deactivate`

(You dont need to, but it makes it easier to run when you are out of the enviroment as I wrote the run.sh file to do it for you)

Then I'm pretty sure you know to install python libraries using `pip`
If not, do the following:

`pip install minecraft_launcher_lib`

This installs `minecraft_launcher_lib` on your virtual enviroment.
(This should be done in the virtual enviroment, so dont exit out of it imeadeatly.)

After all this, you can finally run run.sh using:

`./run.sh`

A bunch of things will pop up in the terminal, but thats just Minecraft 1.8.9 .
